1. Вызов person.name()

    Метод может сгенерировать исключение

    Возвращаемая строка может вызвать std::bad_alloc

2. Оператор << для person.name()

    Оператор вывода может сгенерировать std::ios_base::failure

3. Оператор << для '\n'

    Оператор вывода может сгенерировать std::ios_base::failure

4. Вызов person.grade()

    Метод может сгенерировать исключение

5. Оператор == для сравнения с 10

    Оператор сравнения может быть перегружен и генерировать исключение

6. Вызов person.salary()

    Метод может сгенерировать исключение

7. Оператор > для сравнения с 1000000

    Оператор сравнения может генерировать исключение

    Числовой литерал может вызвать проблемы с компиляцией

8. Оператор ||

    Может вычислять оба операнда (нет short-circuit если первый true)

9. Вызов person.id() в ветке success

    Метод может сгенерировать исключение

10. Вызов save(Status::success, person . id ())

Функция save может сгенерировать исключение

11. Вызов person.id() в ветке failure

    Метод может сгенерировать исключение

12. Вызов save(Status::failure, person . id ())

    Функция save может сгенерировать исключение

13. Исключение при передаче временного объекта person.id() в save()

    Временный объект может быть скопирован, копирование может выбросить исключение

14. Исключение при передаче временного объекта Status в save()

    Копирование перечисления (если это не простой enum)

15. Исключение при уничтожении временных объектов после вызова save()

Д   еструкторы временных объектов могут генерировать исключения

16. Исключение при выходе из функции (раскрутка стека)

    Деструкторы локальных объектов (если бы были) могут генерировать исключения

17. Исключение при преобразовании типа возвращаемого значения person.grade()

    Неявное преобразование к int для сравнения с 10

18. Исключение при преобразовании типа возвращаемого значения person.salary()

    Неявное преобразование для сравнения с 1'000'000

19. Исключение из-за неправильного синтаксиса 1'000'000

    Компилятор может не поддерживать цифровые разделители

21. Исключение при попытке вывести символ новой строки в закрытый поток

    Поток cout мог быть закрыт ранее

22. Исключение при динамическом приведении типов внутри методов

    Если внутри методов используется dynamic_cast
